HTML

  • HTML stands for HyperText Markup Language
  • HTML is used to create the structure of a webpage
  • HTML elements are represented by tags
  • The head element contains information about the webpage

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is used to style the appearance of a webpage
  • CSS can be applied to HTML elements using selectors
  • There are different types of CSS selectors, such as class, id, and element selectors
  • CSS properties are used to define the style of an element, such as color, font-size, and margin
  • CSS can be included in an HTML document using the style element or an external CSS file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the inside of an element

Git

  • Git is a version control system that allows you to track changes in your code
  • git status: checks for any changes and branch we are currently working on
  • git add: adds changes to the staging area
  • git commit: saves changes to the local repository
  • git push: sends changes to the remote repository
  • git pull: retrieves changes from the remote repository
  • git clone: copies a repository from a remote server to your local machine
  • git branch: lists all branches in the repository

JavaScript

  • JavaScript is a programming language that is used to add interactivity to webpages
  • JavaScript can be used to manipulate the content of a webpage, respond to user actions, and communicate with a server
  • JavaScript code is executed by the browser
  • JavaScript can be included in an HTML document using the script element or an external JavaScript file
  • JavaScript variables are used to store data
  • JavaScript functions are blocks of code that can be called and executed
  • JavaScript objects are used to store multiple values
  • JavaScript arrays are used to store multiple values in a single variable
  • JavaScript loops are used to repeat a block of code
  • JavaScript conditionals are used to make decisions in code
  • JavaScript events are used to trigger code based on user actions
  • JavaScript DOM manipulation is used to interact with the HTML elements of a webpage
  • JavaScript AJAX is used to communicate with a server without reloading the page
